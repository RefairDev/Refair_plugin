This plugin is used as a complementary part of the REFAIR Theme. It provide custom posts types and their meta, pdf and xls generations and ui customizations.

== Description ==

1. Public custom posts defined

	* Deposit
	* Provider

2. Taxonomies defined

	* Commune
	* Fournisseur

3. Documents generated by the plugin

	* Archive of an order: xls and pdf of the order, all materials of the order
	* PDF Material sheet
	* Archive of deposit with deposit summary and all linked material sheets
	* Archive of all orders linked to a deposit.

== Usage ==

* Prerequisite
WordPress: > 6.7.2
Woocommerce: > 6.5.2

* Install
1. Upload `refairplugin` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set your API google key in refairplugin settings
